- ? filtra (...) - consente di produrre un elenco filtrato secondo un criterio specifico -

- ? ordina(...) - consente di ordinare l'elenco secondo un criterio specifico -

- ? aggiorna(...) - consente di aggiornare l'elenco specificando l'aggiornamento tramite parametro

- ? somma(...) - consente di sommare campi numerici specificando il campo tramite parametro (es. posti macchina complessivi, posti letto complessivi)

-  ?  ElencoAgriturismi carica( String nomefile) - consente di creare un elenco di agriturismi da un file CSV  specificato come parametro
